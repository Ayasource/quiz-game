QuizMaster Documentation

Documentation

This page outlines how to use the QuizMaster application, its structure, and how each component functions.

Features

  • Accessible controls and navigation
  • Category selection for tailored quizzes
  • Timed and non‑timed quiz modes
  • Score tracking and leaderboard
  • Feedback with SVG‑based indicators

File Structure

index.html — Main app
about.html — About page
docs.html — Documentation page
css/styles.css — Styles
js/app.js — Core logic

Usage

Select a category and start the quiz. Use keyboard or mouse to choose answers. Your score and progress update in real time.
